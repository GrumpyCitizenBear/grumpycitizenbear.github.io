一、所有标签都有的属性

  • 标记 class

1.href
  （1）超链接
  （2）href的取值
      网址
      路径
      伪协议（javascript：代码；、mailto：邮箱、tel：电话）

2.target
  （1）_blank 在空白页面打开
  （2）_self 在当前页面打开
  （3）_top 在顶部打开页面
  （4）_parent 在父页面打开页面
  （5）程序猿的命名 固定窗口xxx（windows的name、iframe的name）
3.download 下载
4.rel=noopener 防止一个bug
