一、所有标签都有的属性

  • 标记 class
  • 格式 style
  • 使元素被编辑 contenteditable
  • 让一个东西看不到 hidden
  • id 在html中是唯一的，可以作为链接锚，通过js或者css改变样式
  • 控制tab键的顺序 tabindex
  • 显示完整内容 title
